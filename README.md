# SmartLab Redes VR - UNB

Este repositório reúne o planejamento técnico do projeto SmartLab Redes VR, desenvolvido para a atividade "Meu Primeiro Ambiente VR – Relatório Técnico", da Residência em TIC 29.

O projeto propõe uma sala de aula inteligente e imersiva inspirada no Lab Redes da Universidade de Brasília. O ambiente virtual foi pensado para apresentar computadores com estética futurista, televisões interativas, um servidor como objeto de estudo e elementos educacionais voltados ao ensino de redes, infraestrutura computacional e tecnologias imersivas.

## Situação do desenvolvimento

Durante a tentativa de uso do Unity Editor, o software apresentou falha de inicialização e encerramento inesperado durante a abertura. Por esse motivo, não foi possível avançar para a implementação prática da cena no editor.

Assim, este repositório organiza a proposta como um planejamento técnico completo, contendo relatório, estrutura prevista do projeto, assets planejados, hierarquia de GameObjects e etapas de execução.
